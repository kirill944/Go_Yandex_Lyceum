# Описание проекта

expression - строка-выражение состоящее из односимвольных идентификаторов и знаков арифметических действий 
Входящие данные - цифры(рациональные), операции "+", "-", "*", "/", операции приоритезации "(" и ")" 
В случае ошибки записи выражения функция выдает ошибку.

# Примеры 

Пример запроса: curl --location 'localhost/api/v1/calculate' --header 'Content-Type: application/json' --data '{"expression": "2+2*2"}'

Запрос:
{"expression": "2+2*2"}
Ответ:
{"result": "6"}
Код: 200

Запрос:
None
Ответ:
{"error": "Internal server error"}
Код: 500
